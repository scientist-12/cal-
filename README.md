# cal

This is a calendar app that I am building 
as a personal project, called cal

I've noticed, over the years, that I've found it 
increasingly difficult to find a calendar 
software or app that does what I want it to, 
without those features being locked behind a
paywall 

So this is my solution.

## features so far
 - Event creation and storage 
 - Events are retrieved by date 
 - A check for upcoming events
